## Profeedback (by Ankicraft)

You finish a study session and Anki just… goes quiet. No reward, no sense of progress, no signal that you did something real 🎯

ProFeedback gives Anki the feedback loop it's missing.

![ProFeedback](https://github.com/cortezmichael594-lgtm/Anki-Profeedback/blob/main/assets/Screenshot%202026-07-15%20130640.png?raw=true)

* 🔊 **Instant sound on correct answers** — a satisfying chime the moment you tap Good or Easy.
* 🎊 **Confetti celebrations** — side-burst when you finish a deck, rain from above when you clear everything; gold confetti + golden heading if the session was perfect.
* 🏷️ **Smart card badges** — see at a glance if a card is new, hard, or one you've failed recently.
* 🎨 **Colorful 3D answer buttons** — vivid, tactile buttons that make every review feel more engaging.
* 🔉 **Custom sounds for every milestone** — separate audio for deck finish, all-done, and perfect sessions.
* ⚙️ **Clean settings panel** — live preview of every sound, all options in one place.
* 🌍 **57 languages** — the interface adapts to Anki's language automatically.

![ProFeedback Settings](https://github.com/cortezmichael594-lgtm/Anki-Profeedback/blob/main/assets/Screenshot%202026-07-15%20130554.png?raw=true)

## HOW TO USE

1. Install and restart Anki.
2. Study normally — sounds and confetti appear automatically.
3. **To add your own sounds:** drop `.mp3` files into `sonidos/correcto`, `sonidos/finalizado/mazo`, and `sonidos/finalizado/todo` inside the addon folder.
4. Configure everything from **Tools → Add-ons → ProFeedback → Config**.

## COMPATIBILITY

Tested on Anki 25.09.x, Windows 11. Should work on macOS and Linux — feedback on other platforms welcome.
